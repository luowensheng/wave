# Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its code of
conduct.

The full text is published, unchanged, at:

  https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project — issues, pull requests, discussions,
Discord, anywhere the project name is on the door — you agree to
follow the Covenant.

## Reporting

If you witness or experience behavior that violates the Covenant,
contact the project maintainers privately at:

> **conduct@TBD** *(replace with a real address before public launch)*

Reports are kept confidential. The maintainers will respond within a
reasonable time frame and follow the enforcement ladder described in
the Covenant.

## Scope

Applies to all project spaces — repository, issues, pull requests,
Discussions, Discord, social media accounts associated with the
project, and any in-person or virtual events where someone is
representing the project.
